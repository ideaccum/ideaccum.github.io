Ideaccum Libraries

Ideaccum LibrariesはJava、Javascript実装時の利用頻度の高い処理をラップしたライブラリなどを提供します。

jQuery Floatcard Plugin

jQuery Floatcard Pluginは、メッセージやブロック要素をコンテンツ上にフロート表示するjQueryプラグインです。

Commons

Commonsは、Java実装における利用頻度の高い操作を簡潔に実装するための各種ユーティリティを提供するライブラリパッケージです。

Library page >>

Commons Config

Commons Configは、外部プロパティリソース定義情報を利用する際の操作を標準化するためのクラスが提供されるパッケージです。

Commons Message

Commons Messageは、外部定義されたメッセージ定義情報を利用する際の操作を標準化するためのクラスが提供されるパッケージです。

Commons Logger

Commons Loggerは、外部定義されたメッセージ定義情報をもとにメッセージコードベースでログ出力を行うためのパッケージです。
