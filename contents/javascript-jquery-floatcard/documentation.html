Documentation

jQuery Floatcard PluginのAPI仕様です。

オプション

autoHide

Type Boolean
Default false

trueを指定した場合、フロート要素表示後に一定時間が経過したタイミングで自動的に表示要素を消去します。
自動消去するまでの時間は、autoHideDelayオプションで任意の時間を指定できます。

autoHideDelay

Type Number
Default 5000

autoHideがtrueの場合に有効なオプションで、自動消去までの時間をミリ秒で指定します。

clone

Type Boolean
Default true

ブロック要素を指定して利用する場合に対象ブロック要素をクローンしてからフロート表示するかを指定します。
フロート表示要素に元の要素をそのまま埋め込む場合にはこのオプションでfalseを指定します。

position

Type "right-top", "right-bottom", "left-top", "left-bottom", "center-top", "center-bottom", "fill-top", "fill-bottom"
Default "right-top"

フロート表示する位置を右上("right-top")、右下("right-bottom")、左上("left-top")、左下("left-bottom")、中央上("center-top")、中央下("center-bottom")、画面上("fill-top")、画面下("fill-bottom")で指定します。
要素の追加方向は上側配置の場合は下方向に、下側配置の場合は上方向に追加されます。

state

Type "default", "fatal", "error", "warn", "info"
Default "default"

フロート表示要素に適用するライブラリが提供する状態スタイルを指定します。
ライブラリが提供する状態スタイルは、ディフォルト("default")、致命的("fatal")、エラー("error")、警告("warn")、情報("info")となります。

styleClass

Type String
Default (undefined)

フロート表示要素に適用する任意のスタイルクラスを指定します。
ライブラリが提供する状態スタイル以外のスタイルを適用したい場合に自身で定義したスタイルクラスを適用することができます。

zIndex

Type Number
Default 2000

フロート表示要素のz-index属性を指定します。
ページレイアウトやCSSフレームワーク等の関連で、個別にz-indexを指定したい場合に利用します。

container

Type Object
Default (undefined)

フロート要素の追加対象コンテナ要素をbody以外の要素に対して行いたい場合に指定します。
パラメータはjQueryオブジェクト又は、セレクタ文字列で指定します。
