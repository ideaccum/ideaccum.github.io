Demo

jQuery Floatcard Pluginの利用方法とその動作デモです。

単純なメッセージ表示

一番シンプルな使用方法は表示したい文字列のみをパラメータとして指定して実行します。
表示位置などはすべて初期値が適用され、右上に追加した順序でメッセージが表示されます。

動作確認
  $.floatcard("単純なメッセージ表示");

ブロック要素を利用した表示

任意のブロック要素のセレクタを指定して取得したjQueryオブジェクトをフロート表示します。
ブロック要素はクローンされた状態埋め込まれますが、オプション(clone)を明示的に指定することで元のブロック要素をそのまま埋め込むことも可能です。

このブロック要素をフロート表示要素の内容として利用します。
ブロック要素内の内容は任意の内容を配置して利用します。

動作確認
  <div id="testBlockElement">
    ...
  </div>
			
  $("#testBlockElement").floatcard();

一定時間で自動消去

autoHideオプション指定によって、ユーザーによる閉じる操作がない場合に一定時間でメッセージが自動消去されます。
自動消去されるまでの時間はautoHideDelayオプションによって、ディフォルト値から任意の時間に変更することが可能です。

動作確認
  $.floatcard("一定時間で消えるメッセージ", {
    autoHide: true, 
    autoHideDelay: 2000
  });

要素の追加方向は上側配置の場合は下方向に、下側配置の場合は上方向に追加されます。

レベルを指定して表示

フロート表示要素はstateオプションによって状態レベルごとのスタイルを提供して表示できます。
状態レベルはディフォルト、致命的、エラー、警告、情報が提供されます。

ディフォルト
致命的
エラー
警告
情報
  $("#testStateDefalut").bind("click", function() {
    $.floatcard("ディフォルト", {
      state : "default"
    });
  });
  $("#testStateFatal").bind("click", function() {
    $.floatcard("致命的", {
      state : "fatal"
    });
  });
  $("#testStateError").bind("click", function() {
    $.floatcard("エラー", {
      state : "error"
    });
  });
  $("#testStateWarn").bind("click", function() {
    $.floatcard("警告", {
      state : "warn"
    });
  });
  $("#testStateInfo").bind("click", function() {
    $.floatcard("情報", {
      state : "info"
    });
  });

任意スタイルクラスを適用して表示

状態レベルとして提供されるスタイル以外の任意のスタイルクラスを適用することができます。
このスタイルはディフォルト状態スタイルが提供されたうえで追加適用されます。

スタイル1
スタイル2
  .testStyle1 {
    color: blue;
  }

  .testStyle2 {
    color: red;
  }
			
  $("#testStateDefalut").bind("click", function() {
    $.floatcard("スタイル1", {
      styleClass : "testStyle1"
    });
  });
  $("#testStateFatal").bind("click", function() {
    $.floatcard("スタイル2", {
      styleClass : "testStyle2"
    });
  });

任意のz-indexで表示

コンテンツページによってフロート表示のz-index調整が必要な場合は、zIndexオプションで指定が可能です。

動作確認
  $("#testZIndex").bind("click", function() {
    $.floatcard("メッセージ", {
      zIndex : 10000
    });
  });

body以外のコンテナに表示

containerオプションによってbodyタグ以外のコンテナ要素配下に表示要素を追加できます。
フロート要素はposition:absoluteで追加される為、配置先要素はposition:staticを指定することは出来ません。

任意のコンテナ要素内部にメッセージ等を追加することができます。
動作確認
  $("#testContainer").bind("click", function() {
    $.floatcard("メッセージ", {
      container : "#testContainerElement"
    });
  });
